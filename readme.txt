This Plugin Helps you add referral rates to individual affiliates integrating with woocommerce product purchase.

== Description ==

**Plugin options :**

1. Individual referral rates can be added to individual affiliates by the wordpress admin.
1. Easily manage the refferal rates for affiliates and the admin can easily change the rates at any moment.

**Requirements :**

* Wordpress( 4.0 or greater )
* [WooCommerce](https://wordpress.org/plugins/woocommerce/ "WooCommerce" )
* [Affiliates Plugin](https://wordpress.org/plugins/affiliates "Affiliates Plugin By itthinx" )


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/Affiliates-Referrals-for-Woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Then open the Dashboard->Referrals.
1. Select an affiliate from the dropdown an set the referral rate below and press save.Thats it.
1. You can edit the referral rate at any time by opening Dashboard->Referrals->Referral Edits
